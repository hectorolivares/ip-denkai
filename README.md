Simple, minimalista y potente. Una herramienta moderna para visualizar tu identidad en la red.

## # CARACTERÍSTICAS
- **Detección Dual**: IPv4 e IPv6 en tiempo real.
- **Geolocalización**: Ubicación precisa y proveedor de servicios (ISP).
- **Hardware Info**: Detección de arquitectura (ARM/X64), marca de CPU y núcleos.
- **Browser Info**: Identificación de navegador, versión y plataforma.
- **Copy-to-click**: Copia tu IP al portapapeles con animaciones fluidas.

## # TECNOLOGÍAS
